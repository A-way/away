## Getting started


### Installation

```bash
go get -u -v github.com/A-way/away
```

### Usage

Run remote:

```
away -rp 8080 -pk "passkey you like"
```

Run local:

```
away -lp 1080 -pk "passkey you like" -ru http://remote-url:8080
```
